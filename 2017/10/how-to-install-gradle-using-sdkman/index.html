Introduction

Gradle is a general purpose dependency management and build automation tool.

Pre-requisites

Java ( version 7 or higher)

Steps

Installation with package Manager

SDKMAN! is a tool for managing parallel versions of multiple Software Development Kits on most Unix based systems. It provides a convenient Command Line Interface (CLI) and API for installing, switching, removing and listing Candidates. Formerly known as GVM the Groovy enVironment Manager, it was inspired by the very useful RVM and rbenv tools, used at large by the Ruby community.

$ `sdk install gradle 4.2.1`

here version is optional. If we don’t specify any version, SDK will fetch the latest stable version.

Once it is done with installation, it will ask to make the recently installed version as default.

You will see output something like this:

Downloading: gradle 4.2.1

In progress...

######################################################################## 100.0%

Installing: gradle 4.2.1
Done installing!

Do you want gradle 4.2.1 to be set as default? (Y/n): y

Setting gradle 4.2.1 as default.

Once done run following command to check Gradle version

$ gradle -v

It should give following output:
